# 🪟 System Transparency

**System Transparency**는 Windows 환경에서 탐색기창을 자동으로 반투명 처리하여, 작업 환경을 깔끔하고 모던하게 만들어주는 유틸리티입니다.  

---

## ✨ 주요 기능

### 1️⃣ 자동 투명화
- Windows 탐색기(`CabinetWClass`, `ExploreWClass`) 창 자동 반투명화  
- 우클릭 메뉴, 추가 메뉴 등 Windows 11 새로운 메뉴 지원  
- 별도 조작 없이 자동으로 적용

### 2️⃣ 사용자 조정 투명화
- **Ctrl + 마우스 휠 클릭** → 현재 활성 창 반투명화  
- **Win + 마우스 휠 클릭** → 현재 활성 창 불투명화  
- **Ctrl + Win + 마우스 휠** → 창 투명도 단계별 조절

### 3️⃣ 트레이 아이콘
- 시스템 트레이에 상주하며 바로 접근 가능  
- 우클릭 메뉴 제공:
  - 프로그램 정보 확인
  - GitHub 페이지 이동
  - 프로그램 종료

### 4️⃣ Windows 11 호환
- 기존 Explorer 창뿐 아니라 새로운 스타일 메뉴에도 투명화 적용  
- 자동 투명화 + 수동 조절 기능 동시 지원

---

## ⚙️ 설치 및 실행
1. 제공된 설치 파일 실행 또는 압축 해제  
2. 트레이 아이콘이 나타나면 자동 투명화 적용  
3. 필요 시 Ctrl / Win 키와 마우스 조합으로 직접 투명도 조절 가능  

---

## ⚠️ 주의 사항
- 일부 프로그램(게임, 특수 그래픽 앱 등)에서는 투명화 적용이 제한될 수 있음  
- 기본적으로 Windows 탐색기와 메뉴 최적화, 그 외 창은 수동 조절 필요
